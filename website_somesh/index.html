HacktoberFest 2021

Hacktoberfest, in its 8th year, is a month-long celebration of open source software run by DigitalOcean. During the month of October, we invite you to join open-source software enthusiasts, beginners, and the developer community by contributing to open-source projects.

  • Prepare and share your project for collaboration
  • Contribute to the betterment of a project via pull request
  • Organize an event
  • Mentor others
  • Donate directly to open source projects

For more information regarding this event checkout their official website- Hacktober Fest 2021

This website has been created to document the developers who contributed to the repositories of BlackTimber Labs as a part of HacktoberFest '21. Checkout the Contributors page to take a look atthe profiles of the contributors!
